Prácticas de Programación Web

Este repositorio contiene el conjunto de ejercicios y prácticas desarrollados para la materia de Programación Web que se hicieron en el salon de clases

Descripción de la Actividad
La actividad consiste en la replicacion de los codes dados del ejercicio 1 al 9 y en base al conocimiento adquirido desarrollar el ejercicio 10 


Índice de Ejercicios
1. **Ejercicio 1:** Estructura básica de una página web.
2. **Ejercicio 2:** Formato y estilos de texto esenciales.
3. **Ejercicio 3:** Creación y comportamiento de enlaces de práctica.
4. **Ejercicio 4:** Gestión e integración de imágenes locales mediante rutas relativas.
5. **Ejercicio 5:** Integración de elementos multimedia y desarrollo de un blog personal.
6. **Ejercicio 6:** Estructuración semántica utilizando secciones de maquetación en HTML5.
7. **Ejercicio 7:** Implementación de listas ordenadas e inactivas aplicadas a recetas.
8. **Ejercicio 8:** Construcción de tablas complejas para la organización de horarios académicos.
9. **Ejercicio 9:** Diseño y validación de un formulario de contacto profesional avanzado.
10. **Ejercicio 10:** Creación de una calculadora conversora de temperatura de Celsius a Fahrenheit utilizando JavaScript funcional.
